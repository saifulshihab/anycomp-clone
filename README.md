## Anycomp Clone

A platform for company registration and management.

## Installation

- Frontend

Navigate to `frontend` directory & run the following commands to install all dependencies.

```js
npm install
```

Create a `.env` file, and add the required environment variables as specified in the [`.env.example`](frontend/.env.example) file. After that, run the following command to run the application.

```js
npm run dev
```

- Backend

Navigate to `backend` directory & run the following commands to install all dependencies.

```js
npm install
```

Create a `.env` file, and add the required environment variables as specified in the [`.env.example`](backend/.env.example) file. After that, run the following command to run the application.

```js
npm run dev
```
